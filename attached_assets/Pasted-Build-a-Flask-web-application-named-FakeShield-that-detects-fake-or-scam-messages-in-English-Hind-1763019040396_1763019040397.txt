Build a Flask web application named “FakeShield” that detects fake or scam messages in English, Hindi, and Hinglish using NLP. 
The app must be 100% deployable on Render with no errors.

✅ Folder structure:
FakeShield/
│
├── app.py
├── requirements.txt
├── runtime.txt
│
├── /templates
│     ├── index.html      → Message input page
│     ├── result.html     → Detection result page
│     └── learn.html      → Learning center
│
└── /static
      └── style.css       → Shared CSS for all pages

✅ Functional requirements:
1. **Flask Backend**
   - Create endpoints:
     • `/` → renders index.html  
     • `/result` → renders result.html  
     • `/learn` → renders learn.html  
     • `/predict` → POST route that accepts JSON: `{ "message": "<text>" }`
   - Use TF-IDF + Naive Bayes classifier trained on a small synthetic dataset with English, Hindi, and Hinglish messages.
   - Perform preprocessing: lowercase, punctuation removal, and stopword filtering.
   - Return JSON response with:
       `{ "prediction": "Likely Scam" or "Safe Message", "confidence": <percentage>, "language": <detected_language>, "reasons": [<list>] }`
   - Detect language using langdetect + heuristic for Hinglish.
   - Log every prediction to `logs.csv` with timestamp.

2. **Frontend (HTML/CSS/JS)**
   - index.html:
       - Title “FakeShield – Scam Message Detector”
       - Textarea for message input
       - Buttons: “Detect Language” and “Check”
       - Display detected language dynamically
       - Bottom nav bar: “Check”, “Result”, “Learn”
       - Use fetch() to call `/predict` and redirect to /result
   - result.html:
       - Card showing prediction result (red for scam, green for safe)
       - Show confidence and reasons list
       - Button to go back to home
   - learn.html:
       - Title “Learning Center”
       - Three informative cards:
           1. Common Scam Patterns
           2. Spotting Red Flags
           3. Essential Safety Tips
       - Keep color palette consistent with index page
   - style.css:
       - Light theme, centered layout
       - Primary accent color: #0f766e
       - Responsive and mobile-friendly

3. **Render compatibility**
   - Add `app.secret_key = "dev-key"`
   - End of app.py:
       ```python
       if __name__ == "__main__":
           import os
           port = int(os.environ.get("PORT", 5000))
           app.run(host="0.0.0.0", port=port)
       ```
   - Add `runtime.txt`:
       ```
       python-3.11.9
       ```
   - Add `requirements.txt`:
       ```
       Flask==3.0.3
       langdetect==1.0.9
       scikit-learn==1.5.1
       numpy==1.26.4
       gunicorn==21.2.0
       ```
